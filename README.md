# NSZ to NSP Converter (C++ / WebAssembly)

Этот проект предоставляет инструмент для конвертации файлов `.nsz` (сжатых NSP-образов Nintendo Switch) в стандартные `.nsp` прямо в браузере с использованием WebAssembly. Конвертация выполняется полностью локально, без отправки данных на сервер.

Алгоритм основан на обратном инжиниринге формата NSZ, используемого в утилите `nsz` (https://github.com/nicoboss/nsz). Он распознаёт заголовок `NSZ0`, извлекает информацию о файлах, распаковывает сжатые блоки (Zstandard) и формирует новый PFS0-контейнер с заголовком `PFS0`.

## Возможности

- Конвертация `.nsz` → `.nsp` в браузере.
- Поддержка как сжатых (Zstd), так и несжатых файлов внутри NSZ.
- Если входной файл уже является NSP (`PFS0`), он просто копируется.
- Автоматическое определение выходного имени (замена расширения).
- Работает полностью на стороне клиента, приватность данных гарантирована.

## Требования для сборки

- [Emscripten SDK](https://emscripten.org/docs/getting_started/downloads.html) (версия 3.1.0 или новее)
- Базовые утилиты сборки (make, cmake не обязательны, достаточно `emcc`)

## Компиляция

```bash
